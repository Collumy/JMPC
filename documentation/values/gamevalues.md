# Игровые значения

Пример использования:
```ts
a = value("location")
b = value("current_health", "default")
```

| **ID**             | **Название**          |
|--------------------|-----------------------|
| `<none>`           | Без цели              |
| `<current>`        | Текущая цель          |
| `<default>`        | По умолчанию          |
| `<default>` | Существо по умолчанию |
| `<killer>`  | Убийца                |
| `<damager>` | Атакующий             |
| `<victim>`  | Жертва                |
| `<shooter>` | Стрелок               |
| `<projectile>`     | Снаряд                |
| `<last>`    | Последняя сущность    |


**Важно:** эти селекторы доступны только для игровых значений. Действия имеют свои собственные селекторы, которые вы должны смотреть на их страницах.
