# Повторитель

## 1. while

| Вариант | Что делает |
|---------|------------|
|  `while True:` | повторять вечно |
| `while <условие>:` | цикл «пока условие истинно» |

```ts
while not player.is_sneaking():
    wait(1)
    n: g`counter` += 1
```

## 2. for … in — встроенные итераторы

Ниже перечислены все готовые итераторы, которые можно использовать после слова `in`.  
Каждый компилируется в соответствующее низко-уровневое действие.

| Итератор | Алиасы |  Описание |
|----------|--------|----------|
| **range** | `range`  | Цикл со счётчиком |
| **diapasone** | `diap`, `diapasone`  | Повторять в диапазоне |
| **list** | `list`, `l`, `mas`  | Проход по списку |
| **map** | `map`, `dict`, `d`  | Проход по словарю |
| **grid** | `grid`, `locs`, `region`  | Проход по региону |
| **adjacent** | `adjacent`, `neighbors`, `nearby`  | Проход по смежным блокам |
| **path** | `path` |  Проход по пути |
| **circle** | `circle` |  Проход по окружности |
| **sphere** | `sphere` |  Проход по сфере |

---

### Примеры «одна переменная»

```ts
for i in range(10):
for loc in grid(location(0,64,0), location(5,64,5)):
```

---

### Примеры «две переменные» 

```ts
for index, item in list(items):
    player::message("Слот ${index} = ${item}")

for key, val in map(prices):
    player::message("${key} стоит ${val}")
```

---

### Полный синтаксис каждого итератора

#### range
```
for переменная in range(количество):
```
| Аргумент | Тип | Описание |
|----------|-----|----------|
| количество | number | сколько итераций |

#### diapasone
```
for переменная in diapasone(начало, конец, шаг):
```
| Аргумент | Тип | Описание |
|----------|-----|----------|
| начало | number | первое значение |
| конец | number | последнее значение (включительно) |
| шаг | number | приращение |

#### list
```
for index, значение in list(список):
```
| Аргумент | Тип | Описание |
|----------|-----|----------|
| список | list | что перебирать |

#### map
```
for ключ, значение in map(словарь):
```
| Аргумент | Тип | Описание |
|----------|-----|----------|
| словарь | dict | что перебирать |

#### grid
```
for переменная in grid(от_локации, до_локации):
```
| Аргумент | Тип | Описание |
|----------|-----|----------|
| от_локации | location | один угол прямоугольника |
| до_локации | location | противоположный угол |

#### adjacent
```
for переменная in adjacent(центр, повороты, включать_центр, паттерн):
```
| Аргумент | Тип | Описание |
|----------|-----|----------|
| центр | location / entity | откуда считать |
| повороты | enum `TRUE/FALSE` | менять ли yaw/pitch |
| включать_центр | enum `TRUE/FALSE` | добавлять саму центральную точку |
| паттерн | enum `CARDINAL/SQUARE/ADJACENT/CUBE` | какие соседи нужны |

#### path
```
for переменная in path(список_локаций, шаг, повороты):
```
| Аргумент | Тип | Описание |
|----------|-----|----------|
| список_локаций | list[location] | опорные точки |
| шаг | number | расстояние между генерируемыми точками |
| повороты | enum `TRUE/FALSE` | выставлять yaw/pitch |

#### circle
```
for переменная in circle(центр, радиус, точек, нормаль_плоскости, стартовый_угол, единицы_угла):
```
| Аргумент | Тип | Описание |
|----------|-----|----------|
| центр | location | центр окружности |
| радиус | number | радиус |
| точек | number | сколько точек сгенерировать |
| нормаль_плоскости | vector | вектор-нормаль к плоскости окружности |
| стартовый_угол | number | смещение по углу |
| единицы_угла | enum `RADIANS/DEGREES` | в чём задан угол |

#### sphere
```
for переменная in sphere(центр, радиус, точек, направление_ротации):
```
| Аргумент | Тип | Описание |
|----------|-----|----------|
| центр | location | центр сферы |
| радиус | number | радиус |
| точек | number | сколько точек |
| направление_ротации | enum `NO_CHANGES/INWARDS/OUTWARDS` | куда смотрит локация |
