# Плейсхолдеры

Плейсхолдеры — это встраиваемые выражения, которые вычисляются прямо в тексте или имени переменной.  
Синтаксис: `$<выражение>`.
Для чисел нужно вводить: `num("выражение")`

Пример:
```ts
player::message("HP: $<hp + 5>")
```


---

#### Автозамены внутри `$< … >`

| Ввод        | Результат     |
|-------------|---------------|
| `math(x)`   | `%math(x)`    |
| `value(id)` | `%val(id)`    |
| `val(id)`   | `%val(id)`    |

---

#### Математические доработки

| Функция            | Ввод            |  Результат         |
|--------------------|---------------- |--------------------|
| `min(a,b,…)`       |  `min(1,2,3,4)` | `min(1, min(2, min(3, 4)))` |
| `max(a,b,…)`       | `max(1,2,3,4)`   | `max(1, max(2, max(3, 4)))`  |
| `clamp(x,min,max)` |  `clamp(2,4,8)` | `min(max(2, 4), 8)`      |

---

#### Работа с переменными

| Функция          | Результат |
|---------------------|--------|
| g\`name\`            | `%var(name)` |
| s\`list\`[i]        | `%index_save(list, i)` |
| i\`dict\`{key}      | `%entry_line(dict, key)` |
| len(smth)         | `%length_local(smth)` |

Примеры:
```ts
player::message("Coins: $<g`coins`>")
player::message("1-й элемент: $<s`items`[0]>")
player::message("Размер инвентаря: $<len(l`inventory`)>")
```
