# Плейсхолдеры

Плейсхолдеры — это встраиваемые выражения, которые вычисляются прямо в тексте или имени переменной.  
Синтаксис: `$<выражение>`.

Пример:
```ts
player::message("HP: $<hp + 5>")
```

Внутри `$< … >` работает мини-язык выражений: короче и мощнее, чем `%math()`.

---

#### Числовые выражения

Короткая форма для чисел:
```ts
num("5 + 3 * 2")   // → %math(5+3*2)
```

---

#### Автозамены внутри `$< … >`

| Ввод        | Станет        |
|-------------|---------------|
| `math(x)`   | `%math(x)`    |
| `value(id)` | `%val(id)`    |
| `val(id)`   | `%val(id)`    |

---

#### Математика без границ

| Функция        | Что делает | Пример |
|----------------|------------|--------|
| `min(a,b,…)`   | минимум из N значений | `min(1,2,3,4)` |
| `max(a,b,…)`   | максимум из N значений | `max(1,2,3,4)` |
| `clamp(x,min,max)` | ограничить диапазон | `clamp(2,4,8)` → `4` |

---

#### Переменные любой области

| JP-запись           | Станет |
|---------------------|--------|
| `g`name``           | `%var(name)` |
| `s`list`[i]`        | `%index_save(list, i)` |
| `i`dict`{key}`      | `%entry_line(dict, key)` |
| `len(smth)`         | `%length_local(smth)` |

Примеры:
```ts
player::message("Coins: $<g`coins`>")
player::message("1-й элемент: $<s`items`[0]>")
player::message("Размер инвентаря: $<len(l`inventory`)>")
```
