# Значения

## 🔤 Литералы

| Тип        | Примеры |
|------------|---------|
| **Текст**  | `"legacy"` `l"legacy"` `m"<#abc>styled"` `p"plain"` `j'{"text":"json"}'` |
| **Число**  | `123` `1.5` `123E5` `num("2^10 + 1")` |
| **Коллекции** | `[1,2,3]` `["a","b"]` `{"x":1, "y":2}` `{1:"one", 2:"two"}` |

---

## 🏭 Фабрики (Built-in конструкторы)

Вызов = новый объект. Все аргументы именованы; `?` = опционально.

| Имя        | Что создаёт | Сигнатура |
|------------|-------------|-----------|
| `location` | позиция мира | `x y z ?yaw ?pitch` |
| `item`     | предмет | `id*` `?name ?count ?lore ?nbt` |
| `sound`    | звук | `sound*` `?volume ?pitch` |
| `vector`   | вектор | `x y z` |
| `particle` | частицы | `particle*` `?count ?spread_* ?motion_* ?material ?color ?size` |
| `potion`   | эффект | `potion*` `?amplifier ?duration` |

```ts
var spawn = location(0, 64, 0);
var sword = item("netherite_sword", name: m"<red>Босс-меч", lore: ["Падает с босса"], count: 1);
```

## 📦 Переменные: 5 уровней области

| Префикс | Область | Жизненный цикл | Пример |
|---------|---------|----------------|--------|
| **нет / `line`** | строка | одна строка / вызов | `i`foo` = 1` |
| `local` / `l` | локаль | событие / игрок | `l`tmp` = 0` |
| `game` / `g` | мир | пока сервер жив | `g`counter++` |
| `save` / `s` | диск | переживает рестарт | `s`top_balance` = 1_000_000` |

Пробелы и спецсимволы экранируются `` `name` ``.
