## Фабрики

Фабрики — функции, которые принимают аргументы и возвращают значение.
Ими создаются предметы, зелья, локации и другие значения, которые можно определить в коде:

Пример использования:
```ts
var a = location(1, 2, 3);
var b = vector(1, 2, 3);
var c = item("stone");
```

### Типы фабрик

Аргументы показывают на типы значений, которые вы должны передать фабрики для получения значения.
Те аргументы, которые заканчиваются на `?`, указывать не обязательно.

| **Имя**      | **Название**   | **Аргументы**                                                                                                                                                                         |
| ------------ | -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `location()` | Местоположение | (`x`: число, `y`: число, `z`: число, `yaw`: число, `pitch`: число)                                                                                                                 |
| `item()`     | Предмет        | (`id`: текст*, `name`: текст, `count`: число, `lore`: список[текст], `nbt`: текст)                                                                                                    |
| `sound()`    | Звук           | (`sound`: текст*, `volume`: число, `pitch`: число)                                                                                                                                    |
| `vector()`   | Вектор         | (`x`: число, `y`: число, `z`: число)                                                                                                                                               |
| `particle()` | Эффект частиц  | (`particle`: текст*, `count`: число, `spread_x`: число, `spread_y`: число, `motion_x`: число, `motion_y`: число, `motion_z`: число, `material`: текст, `color`: текст, `size`: число) |
| `potion()`   | Зелье          | (`potion`: текст, `amplifier`: число, `duration`: число)                                                                                                                             |


# Переменные

Переменные можно использовать **без явного указания области**, либо с явным префиксом.

##### Неопределённые переменные

Если область не указана, она выбирается по контексту:

- в **событиях** — переменная считается **локальной**
- в **функциях/процессах** — переменная считается **строчной** (line)

```ts
varName = 5
`varName` = 10
```

> Обратные кавычки `` `name` `` позволяют использовать пробелы, спецсимволы и динамические имена.

##### Явно определённые переменные

**Строчные (line‑scope)**  
(живут только в рамках текущей строки/вызова)

```ts
i`varName` = 5
line`varName` = 5
```

**Локальные (local‑scope)**  
(живут в рамках события/игрока/контекста)

```ts
l`varName` = 5
local`varName` = 5
```

**Игровые (game‑scope)**  
(общие для всего мира/сервера)

```ts
g`varName` = 5
game`varName` = 5
```

**Сохранённые (save‑scope)**  
(переживают перезапуск, хранятся дольше обычных)

```ts
s`varName` = 5
save`varName` = 5
```
