# Условия

## Базовая форма

```
if условие:
    блок
else if условие:
    блок
elif условие:
    блок
else:
    блок
```

Блоки **обязательно** с отступом (таб / 4 пробела).  
`else if`/`elif` можно писать сколько угодно раз; `else` — не более одного и только в конце.

---

## Отрицание

Поставьте `not` или `!` сразу после открывающей скобки:

```
if (not player(is_sneaking())):
if (!player(is_flying())):
```
